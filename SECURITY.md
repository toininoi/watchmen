# Security Policy

watchmen reads local coding-agent transcripts and can send selected excerpts to
OpenRouter during analysis. Please report security or privacy issues privately.

## Supported versions

Only the latest released version is supported for security fixes before the
first stable `1.0` release.

## Reporting a vulnerability

Use one of these private channels:

- GitHub Security Advisories:
  <https://github.com/firstbatchxyz/watchmen/security/advisories/new>
- Email: security@dria.co

Include the affected version or commit, the command you ran, the relevant
configuration, and clear reproduction steps. Do not include private transcripts
unless they are necessary to demonstrate the issue; redact secrets first.

We will acknowledge reports as soon as practical, triage severity, and publish
fixes with release notes when appropriate.

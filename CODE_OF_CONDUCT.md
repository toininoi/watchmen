# Code of Conduct

watchmen is a technical project. Keep discussion direct, specific, and
respectful.

## Expected behavior

- Be clear about the problem, evidence, and tradeoffs.
- Assume good intent, but do not hide technical disagreement.
- Give reproducible details when reporting bugs.
- Respect people's time, privacy, and boundaries.

## Unacceptable behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without explicit permission.
- Repeated off-topic comments that derail issue or PR discussion.
- Security disclosure in public issues before maintainers have had time to
  respond.

## Enforcement

Maintainers may edit or delete comments, lock threads, close issues, or block
users who make the project harder to use or maintain. Report concerns through
GitHub Security Advisories for security issues, or by email at security@dria.co
for private conduct concerns.
